# Codex App Mint Port

Temporary, local Linux Mint Cinnamon porting flow for the macOS-only Codex App.

This repository does not redistribute the app itself. It rebuilds a local runtime from the official OpenAI `Codex.dmg`.

## Provenance

This repository was generated by Codex, OpenAI's coding agent powered by GPT-5, and then cleaned up for publication after validating the build flow on Linux Mint Cinnamon.

## What it does

`./port-codex-app-mint.sh`:

1. checks local dependencies
2. downloads `Codex.dmg` from the official OpenAI CDN, unless you provide your own DMG
3. extracts the macOS app bundle and `app.asar`
4. rebuilds `better-sqlite3` and `node-pty` for Linux/Electron in an isolated temporary project
5. applies the Linux-specific patches required for the app window and native module loading
6. downloads a local Electron runtime
7. creates a launcher under `./runtime/codex-app/start.sh`
8. installs or refreshes a desktop entry in `/usr/share/applications/codex-desktop.desktop` (mode `755`, requires `sudo`)

## Design constraints

- no updater service
- no `pkexec`
- no system-wide installation of the app itself
- no `sudo npm install -g ...`
- local runtime only, under `./runtime/`
- system `.desktop` launcher in `/usr/share/applications` (app runtime stays under `./runtime/`)

## Requirements

On Linux Mint / Ubuntu:

```bash
sudo apt install curl unzip p7zip-full python3 build-essential
```

You also need:

- Node.js 20+
- `npm`
- `npx`
- an installed Codex CLI available in `PATH`, or reachable through `CODEX_CLI_PATH`

## Usage

Build using the upstream DMG URL:

```bash
cd <project-dir>
bash ./port-codex-app-mint.sh
```

Build from an already downloaded DMG:

```bash
cd <project-dir>
bash ./port-codex-app-mint.sh --dmg /path/to/Codex.dmg
```

Force a clean rebuild:

```bash
cd <project-dir>
bash ./port-codex-app-mint.sh --fresh
```

Skip desktop entry refresh:

```bash
cd <project-dir>
bash ./port-codex-app-mint.sh --skip-desktop-entry
```

Install the desktop entry only for your user (`~/.local/share/applications`):

```bash
cd <project-dir>
bash ./port-codex-app-mint.sh --user-desktop-entry
```

## Generated output

The generated launcher is:

```bash
<project-dir>/runtime/codex-app/start.sh
```

The generated desktop entry is:

```bash
/usr/share/applications/codex-desktop.desktop
```

## Assumptions

- the upstream `Codex.dmg` remains available at the current OpenAI CDN URL
- Electron `40.0.0` remains compatible with the current upstream app bundle
- the upstream bundle still contains `better-sqlite3` and `node-pty` in a form that can be rebuilt on Linux

If OpenAI changes the bundle structure, the patch step may need to be updated.
